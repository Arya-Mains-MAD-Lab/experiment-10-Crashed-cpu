## Aim
To handle images and videos in Flutter applications according to device size using `MediaQuery`.

---

## Procedure
1. Created a new Flutter project.  
2. Imported `Image.asset` and `Image.network` for local and online images.  
3. Used `MediaQuery.of(context).size` to fetch device screen dimensions.  
4. Displayed images with responsive width and height.  
5. Applied `BoxFit` to control how images fit inside containers.  
6. Added text labels to verify responsiveness.
